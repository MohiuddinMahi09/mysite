Real-word Error

A real-word error takes place when a user mistakenly chooses the wrong word among the other alternatives recommended in the suggestion list.

There may occur some real word error where the word is correct on dictionary but have changed the original significance of the sentence which should not be ignored.

The real-word error is a sentence level error which occurs when a word is right from the dictionary but it is not that kind of word which is the suitable word for that sentence.

Real word error pattern in Bangla Language

Serial Number Category Example
1 Vowel Diacritic Change কি->কী
2 Consonant Diacritic Change লক্ষ্য->লক্ষ
3 Grapheme Root Change বাস->বাঁশ, ফলন->ফল
4 Homophone replacement কোন->কোণ
5 Consonant Grapheme Substitute by Semivowel Grapheme in the Medial position ব<ওয়া=খাবার<খাওয়ার, আবার<আওয়ার, শোবার<শোয়ার
6 Vowel character insertion in medial position আন<আইন, গান<গাউন, বাড়ি<বাওড়ি
7 Vowel character deletion in medial position আন<আইন, গান<গাউন, বাড়ি<বাওড়ি
8 Vowel character deletion in medial position কলম<কম, অবতল<অতল, আদর<আর

Here are just a few examples, many more patterns may occur the rea word error.
