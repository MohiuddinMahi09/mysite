Basic word order of a Bangla Sentence

For a language with subject, verb and object, there can be six possible ways in which these elements can.
  1. Subject + Verb + Object (SVO)
  2. Subject + Object + Verb (SOV)
  3. Verb + Subject + Object (VSO)
  4. Verb + Object + Subject (VOS)
  5. Object + Subject + Verb (OSV)
  6. Object + Verb + Subject (OVS)

Grammatical error is a term used in prescriptive grammar to describe an instance of faulty, unconventional, or controversial usage, such as a ​misplaced modifier or an inappropriate verb tense. Also called a usage error. Compare grammatical errors with correctness.

A grammatical error is a part of a text that doesn't follow the rules of standard grammar.

Grammar is the set of structural constraints that determine how different parts of a sentence fit together in a particular language. Grammatical errors occur when an individual breaks normative writing rules for constructing sentences. While some grammar errors are obvious and lead to a sentence that makes no sense, others are more subtle and simply change the meaning of the sentence.
